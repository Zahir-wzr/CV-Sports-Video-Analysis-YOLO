# CV-Sports-Video-Analysis-YOLO
Computer vision based sports video analysis using YOLOv8 for swimming and table tennis.
# Application of Computer Vision in Sports
## Video Analysis of Swimming and Table Tennis Based on YOLO

This project explores the application of computer vision techniques in sports video analysis,
focusing on swimming and table tennis videos using the YOLOv8 object detection model.

---

## Requirements
- Python >= 3.8
- OpenCV
- Ultralytics YOLOv8
- NumPy

pip install -r requirements.txt
